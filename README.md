# Ted - Meu Amigo Virtual 2.4.1

Versão corrigida baseada diretamente na versão funcional enviada pelo usuário.

## Correção importante
- A tela de login foi preservada igual à versão 2.3 funcional.

O reconhecimento de voz depende do navegador. No Android, use preferencialmente o Chrome.
